# HGCByoga Images

This folder contains all images and photos for the Brittany HGCByoga website on Lovable.

## Folder Structure

- **yoga/** - Yoga class photos, poses, and practice images
- **wellness/** - Wellness, meditation, and holistic health images
- **products/** - Product photos and merchandise
- **team/** - Team photos and portraits

## How to Use Images in Lovable

To use any image from this repository in your Lovable website, copy the raw GitHub URL:

**Format:** `https://raw.githubusercontent.com/mindlinerevivalstack/HGCByoga/main/images/[folder]/[filename]`

**Example:** 
- `https://raw.githubusercontent.com/mindlinerevivalstack/HGCByoga/main/images/yoga/class-01.jpg`
- `https://raw.githubusercontent.com/mindlinerevivalstack/HGCByoga/main/images/team/brittany-portrait.jpg`

## Image Guidelines

- **Format:** Use PNG or JPG for best web compatibility
- **Size:** Optimize images for web (typically under 500KB per image)
- **Naming:** Use descriptive, lowercase names with hyphens (e.g., `yoga-class-morning.jpg`)
- **Organization:** Place images in the appropriate subfolder
